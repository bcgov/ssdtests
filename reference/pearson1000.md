# Pearson 1000 Data

An example tibble of 1000 values simulated using a Pearson distribution.

## Usage

``` r
pearson1000
```

## Format

A tbl data frame that includes:

- Conc:

  A numeric vector of the simulate concentrations.

## Examples

``` r
head(pearson1000)
#> # A tibble: 6 × 1
#>    Conc
#>   <dbl>
#> 1  142.
#> 2  146.
#> 3  146.
#> 4  142.
#> 5  149.
#> 6  136.
```
